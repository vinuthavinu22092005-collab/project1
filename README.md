# Project 1: Personal Profile Card

## Learning Objectives

### HTML Concepts:
1. **Semantic HTML**: Use `<header>`, `<section>`, and `<footer>` tags
2. **Images**: Add profile image using `<img>` with proper alt attribute
3. **Links**: Create social media links using `<a>` tags

### CSS Concepts:
1. **Box Model**: Apply margin, padding, and border
2. **Colors and Backgrounds**: Use colors, gradients, and background properties
3. **Typography**: Style fonts, sizes, weights, and text alignment
4. **Flexbox**: Use flexbox for centering and layout

## Instructions

1. Create a profile card structure in `index.html`:
   - Add a header section with profile image
   - Add a section with name and bio
   - Add a footer with social media links

2. Style the card in `styles.css`:
   - Center the card on the page
   - Add padding and border to the card
   - Style the image (make it circular)
   - Style typography for name and bio
   - Use flexbox to arrange social links

## Expected Result
A centered, styled profile card with:
- Circular profile image
- Name and short bio
- Social media links at the bottom
- Nice colors and spacing
